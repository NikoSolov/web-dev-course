Моя яркие проекты

Программа для проведения экспериментов по изучению мозга во время сна

Описание

Данный проект представляет себя графическую программу для проведения экспериментов по изучению работы мозга после пробуждения из быстрой фазы сна.
Программа интегрируется с устройством регистрации ЭЭГ, выставляя на графике временные метки.
Этот проект до сих пор используется и обновляется новыми пытками видами экспериментов.

GitHub репозиторий: github.com/NikoSolov/SleepResearch
